GitHub+hexo，First met

  • 2019-05-13

初识hexo

还是在大二时期，当时在寝室无聊的时候，无意间看到我室友正在学习，恰巧看到他在使用这个东西，当时看了一下觉得有点点意思，但是仅仅是觉得，并没有让我想去学习这个技术的想法。
后来在接触了GitHub之后，感觉这个网站方便的很啊，既能够clone大佬的代码，进行学习，又能够方便的管理好你自己的代码，于是乎就学习了起来。之后在实习的阶段，划水期间，偶然的想弄个自己的博客玩玩，就想起了hexo这个网站，所以无聊之际在网上找起了教程，通过官方文档和他人的博客成功的搭建了一个属于自己的博客。

完善自己的博客

虽然博客搭建好了，但由于工作的缘故导致一直没有时间去完善一下自己的博客，于是乎在项目快上线的时候闲下来了，看到同事的博客之后，忽然想起自己搭建的博客都没去完善过，三分钟热度的我立马搜索着教程撸起袖子干了起来，clone了一个花里胡哨的主题，同事说不好看，但是终究比不过一句我喜欢把，但是在修改之后加载过于慢，下班回到家之后就在搜索博客看看有什么方法，大多数都是说买域名，或者修改电脑上的host文件，第二天上班，同事说加载慢是由于你要加载的东西太多了，后来进到hexo官网进去找过主题，一个一个的点击看看加载的速度，挑选自己所喜欢的进行修改，找到一个蛮喜欢的主题用了起来，修改之后，果不其然，加载比昨天的快了不止一星半点。

为自己的博客加上一点小功能

在搭建好自己喜欢的主题之后，感觉过于单调，于是在主题上添加了一些小功能:
1.增加鼠标点击特效，使用了点击爆炸的特效
2.增加卡通人物（看板娘）
3.增加浏览器网页标题恶搞
4.增加雪花特效
5.设计专属自己的头像logo
6.生成关于我页面
功能实现就不多说了，网上都有教程的，增加这些功能之后，虽然这些花里胡哨的没啥用，但是让自己的博客顺眼了起来，自己开心就好。

over

利用上班的时间来完善自己的博客，这样做也不知道好不好，在完善的过程中突然发现前端以及PS是个很好的东西，各种花里胡哨的东西让你应接不暇，从现在开始就有了属于自己的漂亮博客了。
